## Optional: Forced Language Override

For a complete forced override of English text:

1. Unzip `Assets.zip`
2. Overwrite the original files in:
\Hytale\install\release\package\game\latest\Assets.zip\Server\Languages\en-US
\Hytale\install\release\package\game\latest\Assets.zip\Common\Languages\en-US\avatarCustomization
3. Recreate `Assets.zip`

This ensures all remaining English strings are replaced.
